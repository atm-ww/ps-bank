# ps-banking
兼容 QBCore 和 ESX。

# 依赖
1. [qb-core](https://github.com/qbcore-framework/qb-core) 或 [ESX](https://github.com/esx-framework)
2. [ox_lib](https://github.com/overextended/ox_lib)

# 安装
* 下载发布文件。
* 将资源拖放到服务器文件中。
* 通过 server.cfg 启动资源。
* 将 ps-banking 的 SQL 文件添加到数据库中。
* 重启服务器。
